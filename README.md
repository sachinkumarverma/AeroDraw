# 🎨 AeroDraw Pro — Vivid Studio

AeroDraw Pro is a professional-grade, AI-powered drawing application that lets you create art in the air using simple hand gestures. Powered by MediaPipe's advanced hand-tracking technology, it transforms your webcam into a canvas for fluid, gesture-controlled creativity.

## ✨ Features

- **🚀 Advanced AI Tracking**: High-performance hand detection with stability filtering to eliminate jitter.
- **👐 Two-Hand Zoom & Pan**: Pinch your hands together or pull them apart to zoom in/out, and move both hands to pan across the canvas.
- **🧩 Object Manipulation**: Select and move individual drawings independently using the "Pinch" gesture.
- **🧼 Intelligent Eraser**: Use your palm to erase sections of your artwork. Only triggers when your palm is facing the camera.
- **💎 Premium UI**: Modern glassmorphism interface with a dual-row mobile-responsive control panel.
- **✨ Lumen Brushes**: Dual-layer neon brushes with a vibrant white core and colored glow.
- **⚡ Performance Particles**: Real-time visual feedback with glowing particles that follow your fingertips.

## 🕹️ Gestures

| Action | Gesture |
| :--- | :--- |
| **Draw** | Raise your **Index Finger** and move it to paint. |
| **Erase** | Open your **Full Palm** towards the camera. |
| **Pan / Move** | **Pinch** (Index + Thumb) to grab and move individual items. |
| **Zoom & Pan** | Use **Both Hands**; move them apart to zoom or together to move. |
| **Idle** | Make a fist or hide your hand to stop interacting. |

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+).
- **AI Engine**: [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html).
- **Rendering**: HTML5 Canvas (Dual-layer system).
- **Fonts**: Outfit (Google Fonts).

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser and a webcam.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sachinkumarverma/AeroDraw.git
   ```
2. Open the project folder:
   ```bash
   cd AeroDraw
   ```
3. Run with a local server (like Live Server in VS Code) or simply open `index.html` in your browser.

## 📱 Mobile Support

AeroDraw Pro is fully responsive. On mobile devices, the control panel automatically transforms into a compact, two-row dock at the bottom of the screen to maximize your drawing area.
